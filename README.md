## Preview
https://jacekwilczynski.github.io/masteralb

This is a solution to exercise #1 from
https://drive.google.com/drive/folders/0Bw2hu70L5Ye_VkhvdDZhN3haSlE

## Versions
The `master` branch contains the source as it was made in React. the `gh-pages` branch is a simple html & css version obtained via a customized build process so it looks like React was never used.
